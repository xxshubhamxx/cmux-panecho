cmux로 코딩 에이전트를 스마트폰에서 확인하세요. Mac의 cmux 터미널과 페어링하면 AI 에이전트의 작업을 실시간으로 모니터링하고, 에이전트가 응답을 기다리거나 작업을 완료하는 순간 바로 푸시 알림을 받으며, 어디서든 답장하거나 명령을 실행할 수 있습니다.

cmux는 코딩 에이전트를 위해 설계된 터미널입니다. 세로형 탭, 에이전트 입력 대기 시 울리는 알림 링, 분할 패널, 앱 내 브라우저, 데스크톱 스크립터블 CLI를 갖추고 있으며, iOS 앱은 이 모든 기능의 리모컨 역할을 합니다.

- 실시간 터미널: Mac의 cmux 워크스페이스와 터미널을 스마트폰으로 스트리밍
- 에이전트 알림: 에이전트가 질문하거나 작업을 완료하는 즉시 푸시 알림
- 이동 중 답장: 스마트폰에서 입력 전송, 명령 실행, 워크스페이스 전환
- 음성 입력: 메시지 박스에 받아쓰기
- 보안 페어링: 로컬 네트워크 또는 릴레이를 통해 내 Mac에 안전하게 연결

무료 오픈 소스입니다. cmux가 실행 중인 Mac이 필요합니다 (cmux.com에서 무료로 제공).
