cmux apporte vos agents de code sur votre téléphone. Associez-le à votre Mac exécutant le terminal cmux et regardez vos agents IA travailler en temps réel, recevez une notification dès qu'un agent a besoin de votre attention ou termine une tâche, et répondez ou lancez des commandes de n'importe où.

cmux est le terminal conçu pour les agents de code : onglets verticaux, anneaux de notification quand les agents attendent une réponse, panneaux divisés, un navigateur intégré et une CLI scriptable sur le bureau. L'application iOS est votre télécommande pour tout cela.

- Terminal en direct : diffusez les espaces de travail et terminaux cmux de votre Mac sur votre téléphone
- Notifications d'agents : une notification dès qu'un agent pose une question ou termine un travail
- Répondez en déplacement : envoyez des saisies, lancez des commandes et changez d'espace de travail depuis votre téléphone
- Dictée vocale : dictez dans la zone de message
- Association sécurisée : connectez-vous à votre propre Mac via le réseau local ou un relais

Gratuit et open source. Nécessite un Mac exécutant cmux (gratuit sur cmux.com).
